# bitki-sagligi-teshisi
## Proje Dosyaları
Bu repoda projede kullanılan veri setine erişim bağlantısı yer almaktadır.

## Proje Hakkında
Bu proje, bitki yapraklarının görüntülerini kullanarak bitki hastalıklarını tespit etmeyi amaçlamaktadır.

## Veri Seti
Bitki yaprak görüntülerinden oluşan veri seti, herhangi bir ön işleme uygulanmadan orijinal haliyle paylaşılmıştır. Veri seti farklı bitki türlerine ait sağlıklı ve hastalıklı yaprak görüntülerini içermektedir ve görüntü sınıflandırma çalışmalarında kullanılmaktadır.

GitHub dosya boyutu sınırlamaları nedeniyle veri seti harici olarak barındırılmaktadır.
Kaggle veri seti linki:
https://www.kaggle.com/datasets/abdallahalidev/plantvillage-dataset
